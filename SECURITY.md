# Security Policy

## Supported Versions

Use the latest version of the templates available in the library. Older versions may not be maintained.

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in one of our templates or the platform itself, please report it responsibly.

### How to Report
Please email us at [syedfaizyabhussain07@gmail.com](mailto:syedfaizyabhussain07@gmail.com) with:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

### Response Timeline
- We will acknowledge your report within 48 hours.
- We aim to fix critical issues within 7 days.

Thank you for helping keep the web safe!
