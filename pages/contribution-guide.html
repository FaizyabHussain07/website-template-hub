What You’ll Be Building

Each template must be a ONE-PAGE HTML file (index.html) with the following required sections:

Header (Logo + Nav)
Hero Section
Services / Features
Value Proposition
Interactive (Tabs/Accordion)
Social Proof
Conversion CTA
Footer

Tech Stack

  • Tailwind CSS (CDN)

    For rapid, consistent styling.

  • Google Fonts

    Professional typography.

  • Font Awesome / Lucide

    Modern iconography.

  • Vanilla JS

    For toggles, modals, smooth scroll.

Quality Standards

  • Fully responsive (mobile-first)
  • Clean, well-commented code
  • SEO-friendly structure
  • WCAG 2.1 AA accessible
  • No Lorem Ipsum (Real content)
  • Original work only

Contribution Workflow

1

Choose an Industry

Pick ONE industry from the list and get confirmation from the maintainer.

2

Build Locally

Create 10 single HTML templates locally.

3

Organize Files

Follow the strict naming convention:

templates-library/
├── industry-name-01.html
├── industry-name-02.html
│   ...
└── industry-name-10.html
4

Submit First Template

Submit ONLY 1 template via PR or ZIP for initial review.
